API REST para WhatsApp

WspBox es un servicio API REST para WhatsApp, con soporte para múltiples dispositivos y sesiones concurrentes.

Características Principales

WspBox ofrece una API completa para interactuar con WhatsApp de manera eficiente sin usar recursos pesados como Puppeteer o emuladores de Android.

Alto Rendimiento

Comunicación directa con los servidores de WhatsApp a través de websocket, resultando en menor uso de memoria y CPU.

Múltiples Sesiones

Soporte para múltiples dispositivos y usuarios simultáneamente en la misma instancia.

Mensajes Enriquecidos

Envía texto, imágenes, audio, documentos, videos, stickers, ubicación y contactos.

Webhooks

Configura webhooks para recibir notificaciones en tiempo real de eventos y mensajes.

Verificación de Usuarios

Verifica si los números de teléfono tienen WhatsApp y obtén información del perfil.

Autenticación Simple

Sistema de tokens para autenticación de usuarios fácil y segura.
